RigDial

Hardware remote control via a Contour ShuttleXpress (5 buttons + jog wheel + shuttle ring), driving the same rigctld connection as the Radio tab -- no Flrig involved. The frequency presets below work from the browser alone, with or without the dial plugged in. Needs the hid Python package plus a native hidapi library installed (e.g. port install hidapi or brew install hidapi) -- see the connection status below if the dial isn't showing up.
