WSJT-X Remote

Remote control surface for WSJT-X. Most controls go over its UDP API; a few (Stop, Monitor, Decode, and the band/mode buttons) have no UDP equivalent at all and instead drive WSJT-X's own window directly via AppleScript/Accessibility scripting -- see the hint text below the button row if that stops working (it needs WSJT-X running and this app to have Accessibility permission). Enable Tx and Tune are deliberately left disabled: AppleScript can toggle their on-screen state but doesn't actually key the transmitter, confirmed against rigctld's live PTT reading -- misleading, so not wired up. Rig frequency/band control is separate: see the Radio tab (rigctld).
